Nova AI is a full-stack AI-powered productivity web application designed to help  manage my daily activities, learning goals, and personal growth in a smarter way. The system adapts to the user’s mood and provides personalized suggestions for task management, study planning, and skill development.

The application integrates an intelligent chatbot that assists users with coding help, roadmap generation, and general guidance. Based on user input and mood, the system generates dynamic daily plans, recommends learning resources, and tracks progress over time.

Nova AI also includes features such as a task manager, calendar view, study tracker, and gamification elements to keep users motivated. Additionally, it supports offline or low-connectivity usage by storing essential data locally, ensuring uninterrupted productivity.

This project is built using modern full-stack technologies including React, Node.js, and MongoDB, and integrates AI capabilities to deliver a personalized and adaptive user experience.

Features:

AI Chat Assistant

Mood-Based Recommendations

Smart Daily Planner

Learning Roadmap Generator

Task Management System

Progress Tracking & Gamification

Resource Recommendations

Offline / Low Connectivity Support

Smart Reminders & Notifications

Mood-Based Music Suggestions
